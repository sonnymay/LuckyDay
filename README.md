# LuckyDay

LuckyDay is a simple daily luck guide app built with Expo, React Native, TypeScript, Expo Router, AsyncStorage, and Supabase-ready structure.

The MVP is local-first:
- one-time profile setup
- required face, palm, and handwriting capture
- deterministic daily luck readings
- local feedback storage
- no login, payments, web search, or AI calls

## Run

```sh
npm install
npm start
```

## Verify

```sh
npm run typecheck
npx expo export --platform web
```
